Что такое NPM - Node Package Manager

Это онлайн платформа(репозиторий) для публикации проектов написанный на JS с открытым исходным кодом




*Как app store | play market только для программ написанных на JS

Вторая возможность это - Command Line Client CLI

Инстумент командной строки, который помогает взаимодействовать с онлайн - платформами, такими как браузеры и серверы




*Умеет загружать и устанавливать программы в ваш проект с помощью одной команды *устанавливаеться вместе с NodeJS
